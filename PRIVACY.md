# Aria - Privacy Statement

Aria is a personal AI assistant built for sole use by George Ladikos. 
It is not a public service.

**Data handling:**
- All data (bank balances, transactions, notes, conversations) is stored 
  locally on the user's personal devices.
- No data is shared with third parties beyond the API providers explicitly 
  configured by the user (Anthropic, ElevenLabs, Google, Enable Banking).
- Bank account data fetched via Enable Banking is stored locally only 
  and never transmitted elsewhere.
